IF ANY ITEM FAILS:
──────────────────

1. Check console (F12 → Console) for error messages
2. Look for 404 errors in Network tab
3. Verify file exists in GitHub repository
4. Check path uses: ./ or ../  (not / or absolute path)
5. Verify folder/file names match exactly (case-sensitive)
6. Verify forward slashes used (not backslashes)
7. Try hard refresh: Ctrl+Shift+R (clears cache)
8. Wait 5 minutes (GitHub Pages caches for a minute)
